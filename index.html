Companion mod

TavSync Work in progress

A BG3 Script Extender mod that auto-syncs your party, gear, and game state so you never enter anything by hand. Not yet published — the console-paste bridge currently fills this role.

⚔

Party & Gear

Classes, subclasses, multiclass splits, and all 12 equipped gear slots per character.

★

Abilities & XP

STR/DEX/CON/INT/WIS/CHA scores and experience progress for every party member.

⚑

Game Progress

Current region, act, gold, companions, story milestones, and illithid powers.

How It Will Work

  1. Install BG3 Script Extender
  2. Download TavSync from Nexus Mods and drop the .pak into your Mods folder
  3. Play normally — every save auto-syncs your party data
  4. Click Sync in the strip above or Load File to pull in the latest data

Sync file location: %LOCALAPPDATA%\Larian Studios\Baldur's Gate 3\Script Extender\party_sync.json

Tav works fully without TavSync — the mod just saves you from entering party info manually.
