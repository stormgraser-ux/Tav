TavSync — Live Party Sync for Tav BG3 Companion
=================================================

REQUIREMENTS
- BG3 Script Extender (https://github.com/Norbyte/bg3se)
- Baldur's Gate 3

INSTALLATION
1. Copy TavSync.pak to:
   %LOCALAPPDATA%\Larian Studios\Baldur's Gate 3\Mods\

2. Add TavSync to your mod load order
   (use BG3 Mod Manager or add manually to modsettings.lsx)

3. Launch the game. Open the BG3SE console (F11) and verify you see:
   [TavSync] Server loaded — F6 to sync, auto-sync on save
   [TavSync] Console relay active

USAGE
- Your party syncs automatically on every save (quicksave, autosave, manual)
- Press F6 for a manual sync at any time
- Open https://stormgraser-ux.github.io/Tav and click "Sync" to load your data

WHAT GETS SYNCED
- Party classes, subclasses, levels, multiclass splits
- All equipped gear (12 slots per character)
- Ability scores (STR/DEX/CON/INT/WIS/CHA)
- XP progress
- Current region and act
- Gold
- Recruited companions
- Key story milestones
- Illithid powers

SYNC FILE LOCATION
%LOCALAPPDATA%\Larian Studios\Baldur's Gate 3\Script Extender\party_sync.json

UNINSTALL
Delete TavSync.pak from your Mods folder and remove it from modsettings.lsx.
